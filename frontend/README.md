# React + TypeScript + Vite

This project uses a lightweight React setup powered by Vite, with Hot Module Replacement (HMR) and a basic ESLint configuration.

At the moment, two official React plugins are supported:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react), which uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc), which uses [SWC](https://swc.rs/)
